# HPP-Phthalates-Cardiac
Script Author: Danielle Stevens (danielle_stevens@med.unc.edu)
Script Reviewer: Carson Mowrer

This script was generated to support the analyses and associated tables/figures for the Human Placenta and Phthalates (HPP) Study proposal examining gestational exposure to legacy phthalates and replacement chemicals in association with fetal cardiac function and development. SAS 9.4 and R 4.5.1 were used for analyses. Script is associated with unpublished manuscript with citation:

Stevens DR, Mowrer C, Sinkovskaya E, Welch BM, Landstrom AP, Przybylska A, Calafat AM, Botelho JC, Abuhamad A, Saade G, Buckley JP, Ferguson KK. Legacy Phthalates and Replacement Chemicals Modify Fetal Cardiac Development.
